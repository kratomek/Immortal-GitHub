# 🎮 AntiMicroX – Gamepad guide

## 1. Downloading and Installing AntiMicroX

1. Visit the official project website: [github.com](https://github.com/AntiMicroX/antimicrox) or [antimicrox.com](https://www.antimicrox.com/)
2. Download the installation file for your operating system (Windows, Linux).
3. Install the program in the standard way.

## 2. Launching AntiMicroX

1. Open the **AntiMicroX** program.
2. Connect your gamepad to the computer (USB or Bluetooth).
3. AntiMicroX should automatically detect the connected gamepad.

## 3. Loading the prepared profile

1. In AntiMicroX, click on **File → Load Profile**.
2. Choose: `src/input/gamepad.gamecontroller.amgp`
3. The profile will load, and the gamepad is ready to control the mouse cursor.
