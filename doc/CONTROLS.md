# Controls

This document lists all input controls for **Immortal**, including keyboard, mouse, gamepad, and touch.

## ⚔️ Combat & Abilities
| Key | Action |
|:--:|:--|
| LMB / ⏎             | Attack / Click                |
| Q                   | 1st Ability                   |
| W                   | 2nd Ability                   |
| E                   | 3rd Ability                   |
| R                   | 4th Ability                   |
| D                   | 1st Summon Spell              |
| F                   | 2nd Summon Spell              |
| A                   | Attack Range Toggle           |
---

## 🧭 Movement & Camera
| Key | Action |
|:--:|:--|
| RMB / Arrows        | Player movement               |
| S                   | Stop Movement                 |
| Z                   | Switch Camera                 |
| SPACE               | Center Camera (Champion)      |
| Y                   | Center Camera (Nexus 1)       |
| X                   | Center Camera (Nexus 2)       |
| C                   | Focus Enemy                   |
---

## 📡 Communication & Ping
| Key | Action |
|:--:|:--|
| G / MMB             | Ping                          |
| CAPSLOCK            | Ping Menu                     |
---

## 🕹️ Game Controls
| Key | Action |
|:--:|:--|
| V                   | Change Game Mode              |
| B                   | Recall                        |
| SHIFT               | Mute / Unmute                 |
| 1                   | HUD / Inventory               |
| 2                   | Minimap                       |
| 3                   | Change Skin                   |
| 5                   | Touch Control                 |
| CTRL                | Crouch                        |
---

## ⚙️ System & UI
| Key | Action |
|:--:|:--|
| ;                   | Pause                         |
| ALT                 | Restart Game                  |
| TAB                 | Settings                      |
| \                   | Stats                         |
| 4                   | Key Binding                   |
| 6                   | Sing Up                       |
| NUM7                | Fullscreen                    |
---

## 🎮 Gamepad Controls
**Gamepad guide: [AntiMicroX](AntiMicroX.md)**

| Button              | Action / Description          |
|:-------------------:|:------------------------------|
| A                   | Attack / Click                |
| B                   | Recall                        |
| X                   | Mute / Unmute                 |
| Y                   | Restart Game                  |
| RT                  | Attack / Click                |
| RB                  | Attack Range                  |
| LT                  | Move Player                   |
| LB                  | Stop Movement                 |
| Start               | Pause                         |
| Back                | Settings                      |
| L Stick             | Player Movement               |
| L Stick Click       | Crouch                        |
| R Stick             | Cursor Movement               |
| R Stick Click       | Ping                          |
| D-Pad               | Camera                        |
